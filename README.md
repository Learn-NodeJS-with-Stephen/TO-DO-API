
# A simple TO-DO API

## Steps to install
- Clone the repository
- Install all packages using `npm i`
-


## Endpoints
- Create Task
- Get All Tasks
- Get All Unfinished Tasks
- Get All Completed Tasks
- Get Single Task
- Update Task
- Delete All Completed Tasks
- Delete Single Task
- Delete All Tasks
